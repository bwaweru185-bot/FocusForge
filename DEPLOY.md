# Netlify Deployment Instructions

## 🚀 Quick Deploy to Netlify

### Method 1: Drag & Drop (Easiest)
1. Go to [netlify.com](https://netlify.com)
2. Sign up/login with GitHub, GitLab, or email
3. **Drag and drop** the entire `ai-study-app` folder onto the dashboard
4. Your app will be live instantly with a free URL!

### Method 2: Git Integration
1. Push your code to GitHub:
   ```bash
   git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
   git push -u origin master
   ```
2. Go to [netlify.com](https://netlify.com)
3. Click "New site from Git"
4. Connect your GitHub repository
5. Deploy settings: Leave everything as default
6. Click "Deploy site"

## 📱 Install as Mobile App

After deployment, users can install your app on their devices:

### On Mobile:
- Open the deployed URL in Chrome/Safari
- Look for "Add to Home Screen" or "Install App" prompt
- Or tap the share button → "Add to Home Screen"

### On Desktop:
- Open the deployed URL in Chrome
- Click the install icon in the address bar
- Or go to Menu → "Install FocusForge"
